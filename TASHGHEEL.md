# دليل تشغيل المشروع (خطوة بخطوة)

هذا الملف موجّه لأي شخص هيشغّل المشروع أول مرة، حتى لو ما عندوش خبرة سابقة بـ Laravel. اتبع الخطوات بالترتيب وما تقلق لو حسيت إنها كتيرة - كل خطوة بسيطة لوحدها.

---

## 1. البرامج المطلوبة قبل ما تبدأ

لازم تتثبت على جهازك الأربعة دول:

| البرنامج | الإصدار المطلوب | رابط التحميل |
|---|---|---|
| **PHP** | 8.3 أو أحدث | [windows.php.net/download](https://windows.php.net/download/) |
| **Composer** | أي إصدار حديث | [getcomposer.org](https://getcomposer.org/download/) |
| **Node.js** | أي إصدار LTS حديث (يجيب npm معاه تلقائياً) | [nodejs.org](https://nodejs.org/) |
| **XAMPP** (لتشغيل MySQL) | أي إصدار حديث | [apachefriends.org](https://www.apachefriends.org/) |

### كيف تتأكد إنها متثبتة فعلاً؟

افتح **Command Prompt** أو **PowerShell** (دوّر عليها في قائمة ابدأ) واكتب الأوامر دي وحدة وحدة، بعد كل وحدة اضغط Enter:

```
php -v
composer -v
node -v
npm -v
```

لو كل أمر رجّع لك رقم إصدار (زي `PHP 8.4.23` مثلاً) يبقى تمام. لو ظهرت رسالة زي `'php' is not recognized` يبقى البرنامج ده لسه ما اتثبتش صح أو محتاج تضيفه لـ PATH - أعد تثبيته واتبع تعليمات التثبيت بعناية.

> **ملاحظة لمستخدمي XAMPP:** أحياناً PHP اللي جوا XAMPP يكون إصدار قديم (8.1 أو 8.2) وما يكفيش لهذا المشروع (يحتاج 8.3+). لو عندك نفس المشكلة، ثبّت PHP 8.3+ في مجلد منفصل (مثلاً `C:\php`) وحط مساره في بداية PATH، واستخدم XAMPP فقط لتشغيل MySQL.

## 2. الحصول على المشروع

### الطريقة الأولى: من GitHub

```bash
git clone https://github.com/MGTO21/graduation-project-.git
cd graduation-project-
```

### الطريقة الثانية: من ملف مضغوط (zip)

1. فك ضغط الملف في أي مكان تحب (مثلاً سطح المكتب).
2. افتح Command Prompt أو PowerShell.
3. ادخل لمجلد المشروع بأمر `cd`، مثال:
   ```bash
   cd Desktop\my-project-platform
   ```

من هنا الخطوات نفسها للطريقتين.

---

## 3. تثبيت المكتبات

داخل مجلد المشروع، شغّل:

```bash
composer install
```

هذا الأمر بيجيب كل مكتبات PHP اللي المشروع محتاجها (موجودة أسماؤها في `composer.json`، بيتم تحميلها في مجلد `vendor/`). ممكن ياخد دقيقة أو دقيقتين حسب سرعة النت.

بعدها:

```bash
npm install
```

نفس الفكرة بس لمكتبات الواجهة (CSS/JS)، بتتحمل في مجلد `node_modules/`.

---

## 4. إعداد ملف البيئة (.env)

المشروع محتاج ملف اسمه `.env` فيه إعدادات قاعدة البيانات وغيرها. انسخ الملف الجاهز:

```bash
copy .env.example .env
```

(على Mac/Linux استخدم `cp .env.example .env` بدلاً منه)

بعدين ولّد مفتاح تشفير فريد للمشروع:

```bash
php artisan key:generate
```

الإعدادات الافتراضية في `.env` مضبوطة لقاعدة بيانات اسمها `elearning_platform` بمستخدم `root` بدون كلمة مرور - وهو بالضبط إعداد XAMPP الافتراضي، فمفيش داعي تغيّر حاجة عادةً. لو قاعدة بياناتك عندها كلمة مرور أو اسم مختلف، عدّل هذي الأسطر في `.env`:

```
DB_DATABASE=elearning_platform
DB_USERNAME=root
DB_PASSWORD=
```

---

## 5. تشغيل قاعدة البيانات

1. افتح **XAMPP Control Panel**.
2. اضغط **Start** بجانب **MySQL** فقط (مش محتاج Apache، المشروع هيتشغل بأمر `artisan serve` بدالو).
3. تأكد إن السطر بجانب MySQL بقى أخضر ومكتوب عليه Running.

---

## 6. إنشاء الجداول وتعبئة بيانات تجريبية

```bash
php artisan migrate:fresh --seed
```

هذا الأمر بينشئ كل جداول قاعدة البيانات من الصفر، ويحط بيانات تجريبية جاهزة (حسابات دخول، قسم، مقرر، جدول أسبوعي...) عشان تقدر تجرب المشروع فوراً بدون ما تدخل بيانات يدوياً.

بعدها اربط مجلد التخزين (لازم عشان ملفات المحاضرات المرفوعة تظهر):

```bash
php artisan storage:link
```

---

## 7. بناء ملفات الواجهة (CSS/JS)

```bash
npm run build
```

هذا الأمر بيجمّع ملفات Tailwind CSS والـ JavaScript في نسخة جاهزة للتشغيل (موجودة في `public/build/`). لازم تعمله مرة واحدة على الأقل قبل أول تشغيل، وإلا الصفحات هتفتح بدون تنسيق.

---

## 8. تشغيل المشروع

```bash
php artisan serve
```

هيظهر لك سطر زي:

```
INFO  Server running on [http://127.0.0.1:8000].
```

افتح الرابط ده في أي متصفح. خلي نافذة الـ Command Prompt هذي مفتوحة طول ما انت شغال على المشروع - لو قفلتها السيرفر هيتوقف.

## مشاكل شائعة وحلولها

**"could not find driver" أو خطأ متعلق بـ pdo_mysql**
PHP عندك ناقصه إضافة الاتصال بـ MySQL. افتح ملف `php.ini` (مكانه تقدر تعرفه بأمر `php --ini`) وشيل الـ `;` من أول سطر `extension=pdo_mysql` و`extension=mysqli`، واحفظ وأعد تشغيل الـ terminal.

**"SQLSTATE[HY000] [2002] ... actively refused"**
معناها MySQL مش شغال. ارجع لخطوة 5 وتأكد إنه Running في XAMPP.

**الصفحة بتفتح بدون أي تنسيق (تصميم عادي جداً بدون ألوان)**
نسيت خطوة 7 (`npm run build`)، أو شغّلتها قبل ما تنزل المكتبات بـ `npm install`. رجّع الخطوتين بالترتيب.

**"419 Page Expired" لما تحاول تسجل دخول**
امسح الكاش وأعد المحاولة:
```bash
php artisan optimize:clear
```

**فشل رفع ملفات محاضرات كبيرة (فيديو أكبر من كذا ميجا)**
افتح `php.ini` وزوّد القيم دي (100 و120 ميجا كافيين لمعظم الحالات):
```
upload_max_filesize = 100M
post_max_size = 120M
```
وأعد تشغيل السيرفر بعدها.

**أيقونة صورة مكسورة بدل شعار الجامعة/الكلية**
طبيعي لو صور الهوية (`public/images/brand/*`) ما كانتش موجودة في النسخة اللي وصلتك - المشروع بيشتغل عادي بدونها.

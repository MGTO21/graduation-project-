# منصة التعلم الإلكتروني

مشروع تخرج - كلية علوم الحاسوب وتقانة المعلومات، جامعة النيلين.

منصة تعليمية بسيطة تخدم ثلاثة أدوار: أدمن، محاضر، وطالب. فيها لوحة تحكم للأدمن لإدارة الأقسام والمقررات والمحاضرين والطلاب والجدول الأسبوعي، ولوحة للمحاضر لرفع محاضراته وفتح بث مباشر لطلابه، ولوحة للطالب لمتابعة جدوله ومحاضراته والدخول في البث المباشر والشات مع محاضره وزملائه.

> **أول مرة تشغّل المشروع؟** اتبع [دليل التشغيل التفصيلي (TASHGHEEL.md)](TASHGHEEL.md) - فيه شرح خطوة بخطوة مناسب حتى لو ما عندك خبرة سابقة، وحلول لأكتر المشاكل الشائعة.

## المتطلبات

- PHP 8.3 أو أحدث
- MySQL
- Composer
- Node.js + npm

## التثبيت

```bash
composer install
npm install

cp .env.example .env
php artisan key:generate
```

عدّل بيانات الاتصال بقاعدة البيانات في `.env` (اسم القاعدة، المستخدم، كلمة المرور) حسب جهازك.

```bash
php artisan migrate:fresh --seed
php artisan storage:link
npm run build
php artisan serve
```

## حسابات تجريبية

| الدور | الرقم الجامعي | كلمة المرور |
|---|---|---|
| أدمن | `ADMIN001` | `Admin@123` |
| محاضر | `LEC001` | `Lecturer@123` |
| طالب | `STU001` | `Student@123` |

## ملاحظات

- التسجيل الذاتي معطّل، الحسابات تُنشأ فقط من لوحة الأدمن.
- الدخول بالرقم الجامعي وليس بريد إلكتروني.
- البث المباشر مبني على WebRTC (عبر مكتبة PeerJS) - يعمل مباشرة على `localhost`، وللتجربة من جهازين على نفس الشبكة يلزم HTTPS.
- الشات يعمل بطريقة AJAX Polling بسيطة (بدون WebSocket)، شرح تفصيلي لآلية عمله موجود في [SHARH.md](SHARH.md).
